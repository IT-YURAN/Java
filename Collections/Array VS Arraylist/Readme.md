# Array VS ArrayList

## Array

In Java an Array is a fixed-sized homogenous data structure that stores elements of the same type.
**Array:** Arrays have a fixed size and provide a fat performance.  <br> <br>
` An array can contain both primitive data types as well as objects of a class depending on the definition of the array. 
However, ArrayList only supports object entries, not primitive data types. `
## ArrayList
 ArrayList is a dynamic size, part of the **java collection Framework** an is used for storing objects with built-in methods fo manipulation<br>

**ArrayList:** ArrayList is dynamic in size and provides more flexibility and built-in methods.
